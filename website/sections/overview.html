Project overview

Wildfire risk is a rare-event prediction problem. This project builds a unified statewide monthly dataset and trains models to rank where burning is more likely across California.

🔥

Risk, but usable

Not just accuracy—focus on ranking performance and meaningful validation in highly imbalanced settings.

  • Rare-event framing
  • Spatial + temporal signals
  • Interpretability hooks
🗺️

Geospatial-first

A monthly California grid with consistent feature engineering across multiple public sources.

  • Climate, vegetation, land cover
  • Terrain and fire history
  • Joinable, queryable format
⚡

Built to scale

The pipeline is reproducible and model-oriented, producing train/test and temporally held-out validation artifacts for consistent evaluation.

  • Deterministic runs
  • Efficient transforms
  • Model-ready datasets
