Results

On random train/test evaluation, the tuned gradient-boosting model clearly outperforms the strongest baseline in ranking quality. On temporally held-out data, the same model shows a much weaker minority-event signal.

This pattern suggests the model learns strong in-distribution ranking signal but remains sensitive to temporal shift. For planning workflows, the model is useful as a comparative risk-ranking tool, but it still requires calibration and threshold strategy before high-confidence deployment decisions.
