Methodology

We use a staged modeling strategy to separate signal discovery from final optimization. First, we establish interpretable baselines and failure modes for rare-event detection; then we optimize an imbalance-aware gradient-boosting model using PR-AUC as the primary objective.

1

Baselines First

Baseline experiments include Logistic Regression, Random Forest, and Linear SVM to establish reference behavior and calibration risk. Finalized outputs shown on this site emphasize Logistic Regression (best baseline) and Linear SVM.

2

Extreme Imbalance

Positive fire labels are extremely rare in this statewide monthly setting, so thresholded metrics can be misleading. We therefore prioritize ranking metrics and inspect temporal holdout behavior to evaluate real-world robustness.

3

XGBoost + Optuna

CUDA-accelerated XGBoost is tuned with Optuna (10 trials) using PR-AUC as the optimization objective, then evaluated on both random test splits and a strict 2005–2009 temporal holdout.
